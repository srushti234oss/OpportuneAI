# OpportuneAI

OpportuneAI is a student opportunity recommendation website.

## Features
- Profile setup
- Personalized opportunity recommendations
- Scholarships
- Internships
- Hackathons
- Fellowships
- Skill gap analysis
- Application tracker
- Save opportunities

## Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage

## How to Run
Open index.html in a browser.
